# Фронтенд и бекенд

Данный репозиторий содержит реализации практических заданий и контрольных работ
по дисциплине. Каждая реализация находится на отдельной ветке.

В ветке `main` представлена типовая реализация проекта с TypeScript
зависимостью и утилитами для сборки кода, основной пакетный менеджер - `pnpm`.
